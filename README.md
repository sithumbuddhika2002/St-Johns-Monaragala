📚 Class Attend – St Johns Monaragala
Class Attend is a premium, glassmorphism-inspired attendance and availability management system designed to streamline scheduling for students and educators. It features a robust web dashboard and a native-integrated mobile application.

✨ Key Features
📊 Real-time Heatmaps: Visualize student availability across the entire year with interactive heatmaps.
🏆 "Best Day" Algorithm: Automatically calculates the optimal day for classes based on student responses.
🔐 Secure Authentication: Supports both traditional login and Google OAuth 2.0.
📱 Mobile-First Experience: Includes a Flutter-based mobile app with Biometric (Fingerprint/FaceID) login support.
🌓 Modern UI: Premium Glassmorphism design with full Dark Mode and Light Mode support.
🌐 Multi-language: Seamlessly switch between English and Sinhala.
🛠️ Technical Stack
Frontend: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
Backend: PHP (RESTful API).
Database: MySQL.
Mobile: Flutter / Dart (WebView Bridge).
Auth: Google Cloud Identity & Biometric APIs.
🚀 Getting Started
Import the 

setup.sql
 file into your MySQL database.
Configure your database credentials in 

config.php
.
Set up your Google Client ID for OAuth support.
Deploy to a PHP-enabled web server (WAMP/XAMPP or live hosting).
