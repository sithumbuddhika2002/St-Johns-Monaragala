# 📚 Class Attend – St Johns Monaragala

**Class Attend** is a premium, glassmorphism-inspired attendance and availability management system designed to streamline scheduling for students and educators. It includes a powerful web dashboard and a native-integrated mobile application.

---

## ✨ Features

- 📊 **Real-time Heatmaps**  
  Visualize student availability across the entire year with interactive heatmaps.

- 🏆 **"Best Day" Algorithm**  
  Automatically calculates the optimal day for classes based on student responses.

- 🔐 **Secure Authentication**  
  Supports traditional login and Google OAuth 2.0.

- 📱 **Mobile-First Experience**  
  Flutter-based mobile app with biometric authentication (Fingerprint / Face ID).

- 🌓 **Modern UI**  
  Premium glassmorphism design with full Dark Mode and Light Mode support.

- 🌐 **Multi-language Support**  
  Seamlessly switch between English and Sinhala.

---

## 🛠️ Tech Stack

| Layer       | Technology                          |
|------------|-------------------------------------|
| Frontend    | HTML5, CSS3 (Vanilla), JavaScript (ES6+) |
| Backend     | PHP (RESTful API)                  |
| Database    | MySQL                              |
| Mobile App  | Flutter / Dart (WebView Bridge)    |
| Auth        | Google OAuth 2.0 & Biometric APIs  |
